    Custom CSS snippetsðŸ”—

    Custom CSS allows you to modify Stash's stock style sheets.

    The following is a list of some useful CSS snippets. You may use them by copying-and-pasting them into the Custom CSS editor found in the Settings > Interface Configuration panel or by navigating to 127.0.0.1:9999/settings?tab=interface

    Info

    CSS tweaks usually require flushing the browser's cache or performing a "hard reload". This can be achieved by holding Shift or Ctrl (depending on the browser) while reloading the page.

    Warning

    Future releases of Stash may break some or all CSS tweaks.

    ScenesðŸ”—

    Fit more thumbnails on each rowðŸ”—

    Reduce left and right padding on Scene and Performer grid pages allowing for more thumbnails on each row.

    .grid { padding: 0px !important; }

    Allow longer strings when displaying "Studio as Text" on scene thumbnailsðŸ”—

    .scene-studio-overlay {
    font-weight: 600 !important;
    opacity: 1 !important;
    width: 60% !important;
    text-overflow: ellipsis !important;
}

    Hide scene specs (resolution, duration) from scene cardðŸ”—

    .scene-specs-overlay {
  display: none;
}

    Hide studio logo/text from scene cardðŸ”—

    .scene-studio-overlay {
  display: none;
}

    Make the list of tags take up less widthðŸ”—

    .bs-popover-bottom {
  max-width: 500px
}

    Swap studio and resolution/duration positionsðŸ”—

    .scene-studio-overlay {
  bottom: 1rem;
  right: 0.7rem;
  height: inherit;
  top: inherit;
}

.scene-specs-overlay {
  right: 0.7rem;
  top: 0.7rem;
  bottom: inherit;
}

    Adjust the mouse over behaviour in wall modeðŸ”—

    @media (min-width: 576px) {
 .wall-item:hover::before {
   opacity: 0;  
 }

 .wall-item:hover .wall-item-container {
   transform: scale(1.5);
 }
}

    Don't crop preview thumbnailsðŸ”—

    .flexbin > * > img {
  object-fit: inherit;
  max-width: none;
  min-width: initial;
}

    GlobalðŸ”—

    Change the order of navigation bar buttonsðŸ”—

    Use order values below 0 to move specific buttons to the left of the non-ordered buttons,
    and values above 1 to move them to the right of the non-ordered buttons.

    Before:

    After:

    nav .navbar-nav:first-child {
  display: flex;
  flex-direction: row;
}
div.nav-link[data-rb-event-key="/tags"] {
  order: -2;
}
div.nav-link[data-rb-event-key="/movies"] {
  order: -1;
}
div.nav-link[data-rb-event-key="/scenes"] {
  order: 1;
}

    Hide specific navigation bar buttonsðŸ”—

    Hide links to specific sections from the header.

    /* Example: Hide markers */
div.nav-link[data-rb-event-key="/scenes/markers"] {
  display: none;
}

    Hide the Donate buttonðŸ”—

    .btn-primary.btn.donate.minimal {
  display: none;
}
